## Installation
1. Clone the repo
```bash
git clone https://github.com/hridaya423/newster.git
cd newster
```
2. Install dependencies:
```bash
npm install
```
3. Go to .env:
```bash
NEWS_API_KEY=
GROQ_API_KEY=
```
4. Run it, and go checkout on `https://localhost:3000`
```bash
npm run dev
```
